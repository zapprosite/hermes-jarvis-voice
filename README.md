# Hermes Jarvis Voice

**Voice stack (wake + STT + LLM + TTS + voice loop + persona) para hermes-agent**

## Install

```bash
pip install hermes-jarvis-voice
hermes_jarvis_voice_bootstrap  # se disponível
```

## Compatibilidade

- Python 3.11+
- hermes-agent >= 0.16.0
- Ubuntu 22.04+
